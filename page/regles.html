Règles du Rugby

Règles de base

  • Le rugby se joue entre deux équipes de 15 joueurs.
  • Le but est de marquer des points en aplatissant le ballon dans l'en-but adverse.
  • Le ballon peut être passé en arrière ou latéralement, mais jamais en avant.
  • Les joueurs peuvent plaquer l'adversaire pour récupérer le ballon.

Comment fonctionnent les points au rugby

  • Essai : 5 points (Action de jeu)
  • Transformation : +2 points (Coup de pied)
  • Pénalité : 3 points (Coup de pied)
  • Drop : 3 points (Coup de pied)
  • Essai de pénalité : 7 points (Décision arbitrale)

Les différents types de carton

  • Jaune : Exclusion temporaire (10 minutes)
  • Rouge : Exclusion définitive (Reste du match)
  • Bleu : Sortie médicale/Commotion (Définitive mais remplaçable)
  • Blanc : Exclusion technique/Amateur (10 minutes)
